## 📌 Project Overview

This project focuses on analyzing Black Friday sales data to uncover customer shopping patterns and generate meaningful business insights. The aim is to understand how customers behave during large-scale sales events and how different factors influence their purchasing decisions.

---

## 🎯 Objectives

* Analyze customer purchasing behavior
* Segment customers using clustering techniques
* Discover relationships between product categories
* Detect anomalies such as high spenders
* Present insights through an interactive dashboard

---

## 🧹 Data Preprocessing

* Handled missing values in product categories
* Encoded categorical variables (Gender, Age, etc.)
* Normalized numerical data
* Cleaned and structured dataset for analysis

---

## 📊 Exploratory Data Analysis (EDA)

* Visualized purchase patterns by age and gender
* Identified popular product categories
* Analyzed relationships between features using charts
* Created interactive visualizations for better understanding

---

## 🤖 Advanced Analysis

### 🔹 Clustering

* Applied clustering techniques (e.g., K-Means)
* Grouped customers based on purchasing behavior
* Identified different customer segments

### 🔹 Association Rule Mining

* Used Apriori algorithm
* Found frequent product combinations
* Identified cross-selling opportunities

### 🔹 Anomaly Detection

* Detected unusually high spending customers
* Identified outliers using statistical methods
